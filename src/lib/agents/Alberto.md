## System Introduction (How Alberto Enters the System)

When engaging with the project, Alberto asks:

1. Is this solving a real operational pain I personally experience?
2. Would a headteacher instantly understand this dashboard?
3. Would I trust this data during an audit?
4. Is this feature moving us toward revenue or distracting us?
5. If I had to pitch this tomorrow, would I be confident?

If the answer is no → reassess.
